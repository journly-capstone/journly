## Description
Journly is a web application for teachers, where teachers can create daily posts of gratitude, share them with others, and search through/save mindfulness books for personal use, or use in the classroom.

#### Built With:
* Java
* Spring Boot
* HTML
* CSS
* JavaScript
* jQuery
* Bootstrap
* Google Books API
* Forismatic API
